Intégration #1

Définition

L'intégration est la transposition du web désign en code informatique. Contrètement, le développeur front-ent reprend la maquette validée du site et la transforme en language HTML, CSS et JavaScript via un éditeur de texte. Chaque
choix graphique du designer sera transformé en ligne de code. On pourrait utiliser la métaphore suivante : le designer parle en français à un ordinateur, or cet ordinateur ne comprend que l'anglais, le développeur sera donc en charge de
traduire le message du designer en anglais pour que l'ordinateur puisse le comprendre. On parle donc bien de language!
